# 🗓️ Sistema de Reservas con Google Calendar

Aplicación web full stack para la gestión de citas, con integración directa a Google Calendar. Permite a los usuarios crear, visualizar y eliminar reservas de manera eficiente y centralizada.

---

## 🚀 Tecnologías utilizadas

### 🔧 Backend
- Node.js
- Express
- PostgreSQL (Neon)
- Google Calendar API

### 🎨 Frontend
- Vue.js
- Tailwind CSS
- FullCalendar
- GSAP (animaciones)

---

## ⚙️ Funcionalidades principales

- 📅 Creación de citas sincronizadas con Google Calendar  
- 🔍 Visualización de eventos en calendario interactivo  
- ❌ Eliminación de citas en base de datos y Google Calendar  
- ⏰ Validación de disponibilidad (evita horarios duplicados)  
- 📱 Interfaz responsive (mobile-first)  
- 🎨 Animaciones modernas con GSAP  

---

## 🧱 Arquitectura

Frontend (Vue) → Backend (Express API) → Base de datos (PostgreSQL - Neon) → Google Calendar API
